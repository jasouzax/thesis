World Navigation Hat

Development of a Wearable Navigation Aid using AIoT for the Visually Impaired

Project Overview

Visual impairment is a global health problem that significantly affects individual’s daily lives. This project develops a wearable navigation hat integrated with AIoT to assist Visually Impaired People (VIP).
